Moved to new blog

Posted: Wed, 21 Sep 2011

I just moved my blog architecture to Hyde. Hyde is a static site generator similar to Jekyll, but written in python and uses Jinja2 templates.

I’m now adapting to Hyde and extending it to my needs. What I have developed so far:

  • A github gist plugin, which takes inline code blocks and creates a gist for your blog post. The gist is then embedded into the appropiate spots in the blog page. On subsequent edits the gist is changed accordingly.

    This has two benefits: nice syntax highlighting ala github and easy forking of any code I may post here.

    The source code is not quite ready for public consumption, but will definitely appear in my repositories or as a gist itself ;-)
