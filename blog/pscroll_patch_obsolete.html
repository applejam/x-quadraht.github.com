pscroll patch obsolete

Posted: Fri, 23 Sep 2011

Peter Hutterer has just sent a few patches to xorg-devel@lists.x.org which make my series of pscroll patches to xorg obsolete:

  • He uses a new valuator class to distinguish “scrolling valuators” from normal valuators. The XInput extension 2.1 (only in Peter’s repositories) contains methods to query scrolling valuators (see his blog post)
  • Valuator values are reported in double precision. My patch used integer values (in hardware resolution) and specified the scaling in another integer field.
  • The whole no-integration part of my patches was not included, the valuators just accumulate their values. Applications have to track and detect changes themselves. I’m not sure how to feel about that. But it’s surely more consistent with the old X.org code.

The patch series is still missing a few parts, like wheel emulation in evdev and synaptics support. I will try to contribute there and develop a Qt4 patch in parallel.

I will do a follow-up blog post about getting Peter’s code running on your machine with jhbuild.
